Sending logs

If you're seeing this page, you probably had a problem. Sorry about that, and thanks for reporting! The instructions below are for enabling logging. This allows me to take a look at what's happening in your machine and help you with your issue.

Thanks for helping out!

Mac OS X

Open a Finder window, press Shift + CMD + G. Paste this: ~/Library/Application Support and press Enter. Find the folder called Aether and go in. Open UserProfile, make sure Aether is closed (i.e. not in background) and open UserProfile.json with a text editor.

Change the debugLogging variable to true, save the file and close. This will make it start keeping logs at 'Logs' folder of the same directory. Restart Aether, and let it run for a few minutes. Then send me the log files created at burak@nehbit.net. If you want more security, here's my public key.

Windows

Open ‘My Computer’, write C:\ProgramData to the location bar and press Enter. Find the folder called Aether and go in. Open UserProfile, make sure Aether is closed (i.e. not in background) and open UserProfile.json with a text editor.

Change the debugLogging variable to true, save the file and close. This will make it start keeping logs at 'Logs' folder of the same directory. Restart Aether, and let it run for a few minutes. Then send me the log files created at burak@nehbit.net. If you want more security, here's my public key.

Linux
