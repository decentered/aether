Open this website in a desktop computer to download Aether, or join the mailing list, so you'll be reminded.

Instructions

Mac

Download and install the new version from the button above. Drag it to Applications as requested. Click 'Yes' when asked if you want to overwrite the existing version.

Windows

Uninstall via the uninstaller provided. Then download and install the new version from the button above.

Linux (Ubuntu / Debian)

Install the deb package provided. It will update the current installation of Aether on your computer.

If you end up having problems, remove the application entirely by doing a sudo apt-get remove aether on a terminal screen, and then install the deb package provided.
